1. What is the difference between map(), filter(), forEach(), find()?

Ans: map() creates a new array with modified elements. filter() creates a new array with only certain elements that satisfy a condition. forEach() executes a function for each element of an array. find() returns the first element of an array that satisfies a condition.

2. What is the difference between Local Storage, Session Storage & Cookie?

Ans:
Local storage: If any data is saved in local storage, that data remains forever, until the data is manually deleted.
Session Storage: If any data is kept in session storage then that data will remain until the browser is closed or the specific tab is not closed.
Cookie: When a user visits a website, the server sends a cookie to the user's browser, which then stores it on the user's computer. The cookie contains information about the user's visit to the website, such as login information, preferences, and browsing history.

3. What is an Event Loop? And how does work it?

Ans: A crucial part of JavaScript's asynchronous programming style is the Event Loop, which enables code to run without stopping and effectively manage I/O activities. Because of the Event Loop, which manages the processing of several concurrent events on a single thread, JavaScript is frequently used to create scalable and effective web applications. The sole task of the event loop is to keep track of the call stack and callback queue. The Event Loop will pull the first event from the queue and put it to the Call Stack, which will then execute it if the Call Stack is empty.

Ans:
quizTimer(): The 'quizTimer' function takes a single parameter called 'dismiss'. This function is starting a timer and displays the countdown in minutes and seconds on an HTML element with an ID of "count". If the 'dismiss' parameter is 'true', the function clears the interval of the timer and stops the countdown. Otherwise, it continues the countdown by incrementing a variable called 'count' every second. The function calculates the minutes and seconds of the countdown by using the 'Math.floor' function to convert the total number of seconds into minutes and then take the remainder to calculate the seconds. It then checks the length of the minutes and seconds to ensure that they are two digits, and if not, it prepends a 0 to the value. The function updates the text content of the HTML element with the ID of 'count' to display the current countdown in minutes and seconds with the 'sec' unit in a subscript. The function also adds a CSS class to the HTML element to change the text color every 60 seconds. If the 'count' is divisible by 2, the text color changes to "text-red-700" and if not, it changes to "text-red-300".
chooseQuiz(): The chooseQuiz function takes two parameters': index' and 'givenAns'. The index parameter is an integer value that represents the index of the quiz in the quizData array, while the givenAns parameter is the answer given by the user for that quiz. The function first checks if the quiz has already been answered by searching for its ID in the answers array. If it is already answered, it updates the answer with the new answer given by the user. Otherwise, it adds the quiz with the user's answer to the answers array. After updating or adding the quiz, the function calls displayAnswers() function to display the updated answers.
